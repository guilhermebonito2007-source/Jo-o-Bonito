# 🚀 QUICK START - Loja Sende

## 📋 Em 5 Passos

### 1️⃣ Terminal (Backend)
```bash
cd backend
npm install
```

### 2️⃣ Inicialize a Base de Dados
```bash
npm run init-db
```

✅ Admin criado:
- Email: `admin@loja.pt`
- Senha: `Admin@123`

### 3️⃣ Inicie o Servidor
```bash
npm run dev
```

Verá:
```
✓ Conectado à base de dados SQLite
🌐 Servidor rodando em http://localhost:3000
```

### 4️⃣ Abra no Browser

**Admin Panel:**
```
http://localhost:3000/admin/admin-login.html
```

**Perfil de Utilizador:**
```
http://localhost:3000/prefil/prefil.html
```

### 5️⃣ Faça Login
```
Email: admin@loja.pt
Senha: Admin@123
```
